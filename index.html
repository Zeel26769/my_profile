HealthChecker

AI/ML Health Prediction Tool

Role

Full Stack Developer

Timeline

March 2023 - May 2023

Technologies

React Node.js MongoDB Scikit-learn D3.js

Project Overview

HealthChecker is an AI-powered health prediction tool that helps users assess their health status based on various parameters. The application uses machine learning models to predict potential health risks and provides personalized recommendations.

Key Features

  • Interactive health assessment questionnaire
  • AI-powered health risk prediction using Scikit-learn
  • Real-time data visualization with D3.js
  • User authentication and profile management
  • Responsive design for all devices
  • Secure data handling and privacy protection

Technical Implementation

The application is built using the MERN stack (MongoDB, Express.js, React, Node.js) with additional Python services for machine learning. The frontend is developed with React and Tailwind CSS, providing a modern and responsive user interface. The machine learning models are trained using Scikit-learn and served via a REST API.

Frontend

  • • React.js with Hooks and Context API
  • • Tailwind CSS for styling
  • • D3.js for data visualization
  • • Axios for API requests

Backend

  • • Node.js with Express.js
  • • MongoDB with Mongoose ODM
  • • JWT Authentication
  • • Python Flask for ML services

Challenges & Solutions

Challenge: Real-time Data Processing

Processing health data in real-time required efficient algorithms to ensure quick response times.

Solution: Optimized ML Pipeline

Implemented feature engineering and model optimization techniques to reduce prediction time by 60%.

Challenge: Data Visualization

Creating intuitive visualizations for complex health data that users could easily understand.

Solution: Interactive Dashboards

Developed interactive dashboards with D3.js that allow users to explore their health metrics in detail.

Full-Stack MERN Application

Role

Full Stack Developer

Technologies

Project Overview

A full-stack MERN application that monitors Amazon product prices in real-time. Users can paste any Amazon product URL and the application scrapes the current price, tracks price history over time, and sends email notifications when the price drops below a target threshold.

Key Features

  • Real-time Amazon product price scraping
  • Price history tracking with visual charts
  • Email notifications on price drops
  • Responsive UI with search and filtering
  • Automated cron-based price checks
